# PRD: UI/UX 체크리스트 (prd_checklist)

> 문서 버전: 1.0 | 최종 수정: 2026-03-03 | 기능 코드: F-04

---

## 1. 기능 개요

게임 인터페이스의 품질을 체계적으로 점검할 수 있는 **7개 카테고리, 37개 항목**의 체크리스트입니다. Lithium Design System 기반으로 게임 UI에 맞게 재구성했습니다.

---

## 2. 체크리스트 카테고리 및 항목

### 2.1 정보구조 (IA) — 5개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 게임 콘텐츠가 논리적 계층 구조로 정리되어 있는가 | 필수 | 로비→모험→던전→보스 등 단계별 흐름 |
| 2 | 내비게이션이 2-3 depth 이내로 유지되는가 | 필수 | 깊은 메뉴는 플레이어 이탈의 원인 |
| 3 | 주요 기능에 3번 이내 터치로 접근 가능한가 | 필수 | 핵심 액션(전투, 인벤토리)의 접근성 |
| 4 | 뒤로가기/홈 버튼이 항상 접근 가능한가 | 필수 | 어디서든 이전 화면이나 로비로 복귀 |
| 5 | 장르 관행에 맞는 용어와 구조를 사용하는가 | 권고 | RPG의 인벤토리, FPS의 로드아웃 등 |

### 2.2 레이아웃 (Layout) — 6개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 핵심 HUD 요소가 화면 가장자리에 배치 | 필수 | HP/MP, 미니맵, 스킬바 등 |
| 2 | 중요 버튼이 엄지 닿는 영역에 있는가 (모바일) | 필수 | 피츠의 법칙 적용 |
| 3 | 게임플레이 영역과 UI 영역이 명확히 구분 | 필수 | 게임 화면을 가리지 않는 UI |
| 4 | 다양한 해상도/비율에서 정상 표시 | 필수 | 16:9, 18:9, 21:9 대응 |
| 5 | 시각적 위계가 명확한가 | 권고 | 크기, 색상, 대비 |
| 6 | 그리드 시스템에 맞춰 정렬 | 권고 | 일관된 간격과 정렬 |

### 2.3 인터랙션 (Interaction) — 6개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 모든 버튼/인터랙션에 시각적 피드백 | 필수 | 누름, 호버, 비활성 상태 |
| 2 | 로딩 시간에 적절한 표시 | 필수 | 400ms 이상 걸리면 피드백 필수 |
| 3 | 되돌릴 수 없는 액션에 확인 다이얼로그 | 필수 | 아이템 삭제, 강화 등 |
| 4 | 드래그 가능한 요소 시각적 표시 | 권고 | 인벤토리, 스킬 배치 |
| 5 | 애니메이션 속도 적절성 (200-400ms) | 권고 | 너무 빠르면 인지 불가 |
| 6 | 터치 타겟 최소 44x44px | 필수 | 피츠의 법칙 기반 |

### 2.4 시각 디자인 (Visual) — 5개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 장르 세계관에 맞는 시각적 테마 | 필수 | RPG→판타지, FPS→밀리터리 |
| 2 | 아이템 등급별 색상 코드 일관성 | 필수 | 일반(회), 고급(녹), 희귀(청), 에픽(보), 전설(주황) |
| 3 | 텍스트 가독성 (명암 대비) | 필수 | WCAG AA 기준 4.5:1 이상 |
| 4 | 아이콘 직관성과 일관된 스타일 | 권고 | 크기, 굵기, 스타일 통일 |
| 5 | 색상만으로 정보 전달하지 않는가 | 권고 | 색각 이상 고려 |

### 2.5 콘텐츠 (Content) — 4개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 텍스트가 간결하고 명확한가 | 필수 | 불필요한 설명 최소화 |
| 2 | 오류 메시지가 해결 방법 안내 | 필수 | "연결 확인 후 재시도" |
| 3 | 숫자 포맷 일관성 | 권고 | 1,234,567 또는 1.2M |
| 4 | 중요 수치 변화량(+/-) 표시 | 권고 | 장비 교체 시 스탯 변화 |

### 2.6 접근성 (Accessibility) — 4개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | UI 스케일 조절 옵션 | 권고 | 50~200% 조절 |
| 2 | 색각 이상 모드 | 권고 | 적록/청황 대응 |
| 3 | 자막/캡션 옵션 | 권고 | 대화, 효과음 자막 |
| 4 | 키 리바인딩 | 권고 | 모든 조작 재설정 |

### 2.7 성능 (Performance) — 3개 항목

| # | 항목 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | UI 애니메이션 60fps 유지 | 필수 | 프레임 드랍 없음 |
| 2 | 스크롤 목록 가상화 | 권고 | 100개+ 항목 최적화 |
| 3 | UI 에셋 아틀라스 최적화 | 권고 | 드로우콜 최소화 |

## 4. 상태 관리

### 4.1 체크 상태 저장

```javascript
// localStorage 키: 'gameui-checklist'
// 값: JSON 객체 — 인덱스(전체 항목 기준)를 키, true를 값으로 저장
{
    "0": true,   // 첫 번째 항목 체크됨
    "3": true,   // 네 번째 항목 체크됨
    "15": true   // 열여섯 번째 항목 체크됨
}
```

### 4.2 진행률 계산

```javascript
// 전체 체크박스 중 체크된 비율을 퍼센트로 계산
const pct = Math.round((checked / total) * 100);
// 프로그레스 바 width와 카운터 텍스트 업데이트
```
